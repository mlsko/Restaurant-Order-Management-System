# Restaurant-Order-Management-System

Webová aplikácia určená na spracovanie a správu dát uložených v databáze MySQL. Aplikácia načítava údaje, ktoré sú do databázy nahrávané prostredníctvom mobilnej aplikácie, a umožňuje ich prehliadanie a ďalšie spracovanie vo webovom prostredí. Riešenie demonštruje prácu so serverovou logikou, databázou a dynamickým generovaním webového obsahu.

Použité technológie:
PHP – serverová logika a spracovanie dát
HTML5 – štruktúra webovej aplikácie
CSS3 – vizuálny dizajn a štýlovanie rozhrania
MySQL – relačná databáza na ukladanie a správu dát
Apache / XAMPP – lokálny webový server pre spustenie aplikácie

Funkcionality
Načítanie dát z databázy MySQL
Zobrazenie údajov vo webovom rozhraní
Dynamické generovanie obsahu pomocou PHP
Spracovanie a manipulácia s dátami
Možnosť filtrovania / prehľadávania údajov 
Prepojenie s mobilnou aplikáciou cez spoločnú databázu
Základná správa dát (CRUD operácie )
